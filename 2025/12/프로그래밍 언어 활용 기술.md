### 비교 연산자
#### 같음연산자(Equality Operator)

• A == B : A, B가 같다면 true, 같지 않다면 false를 반환한다.

• A != B : A, B가 같다면 false, 같지 않다면 true를 반환한다
```csharp
// 같음 / 다름
result = a == b;
result = a !=b;

```

#### 비교연산자(Comparison Operator)

• A > BI: A가 B 초과라면 true, 아니라면 false를 반환한다

• A < B : A가 B 미만이라면 true, 아니라면 false를 반환한다

• A >= B : A가 B 이상이라면 true, 아니라면 false를 반환한다

• A <= B : A가 B 이하라면 true, 아니라면 false를 반환한다
```csharp
int a = 5;
int b = 66;

Console.WriteLine(a > b); // false
Console.WriteLine(a > b); // false
Console.WriteLine(a > b); // false
```
```csharp
int a = 5;
int b = 66;

bool result = a > b;

Console.WriteLine(result) ;
```
```csharp
int a = 5;
int b = 66;
bool result;

// 초과 | 미만
result = a > b;
result = a < b;

// 이상 / 이하
result = a >= b;
result = a <= b;

Console.WriteLine(result);
```

#### 논리연산자

두개 이상의 bool타입 데이터에 대한 논리 연산을 수행한다.

크게는 사용 목적에 따라 Bool 논리연산자, 조건부 논리연산자로 나눠볼 수 있다.

다만, 조건부 논리연산자 쪽이 효율적으로 형태로 동작하기 때문에 자주 쓰이는 방식이다.

그럼에도 불구하고 Bool 논리연산자는 이진수 타입을 연산해야 할 때도 사용되므로 알고
있어야 할 개념이다.

연산자는 기본적으로 이진 형태의 데이터, bool 타입에 대해
수행하는 것이 일반적이다. 연산자의 종류는 아래와 같다.

• NOT(!) : 제시된 하나의 조건에 대해 반대로 출력

• AND ( &) : 제시된 데이터들이 모두 true일때만 true 출력

• OR ( | ) : 제시된 데이터들 중 하나 이상이 true일 때true 출력

• XOR (^) : 제시된 데이터들이 서로 같지 않을 때 true 출력

```csharp
bool a = true;
bool b = false;
bool c = true;
bool d = false;

bool result = a | b | c | d;

Console.WriteLine(result);
```

##### 조건부 논리 연산자

마찬가지로 bool 타입에 대한 논리연산을 수행한다.

• AND ( && ) : 주어진데이터들이 모두 true일 때만 true출력

• OR ( || ) : 주어진 데이터들 중 하나 이상 true라면 true출력

하지만 위의 Bool 논리연산자와의 차이점은, Bool 논리연산자는 실제로 주어진 모든 데이터에 대한 논리연산을 수행하지만, 조건부 논리 연산은 이후의 논리연산을 수행할 필요가 없다면 수행하지 않는다.
