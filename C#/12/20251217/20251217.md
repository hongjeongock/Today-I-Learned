# 소코반 구현
오늘은 강사님이 구현하시는 C# 콘솔 게임(소코반 형태)을 그대로 따라 구현해보는 연습을 해봤다.

플레이어 이동, 폭탄 밀기, 목표 지점 판정 등의 로직을 함수 단위로 나누어 작성했고, 구현 도중 TryPushBomb, PrintMap, GetOriginTile 등의 함수에서 다음과 같은 오류가 발생

![alt text](C#/12/20251217/image.png)

함수 이름은 맞게 작성했다. 근데도 컴파일 오류가 발생하는것이다...

단순 오타겠지 하고 계속 오타 검토를 하며 함수 부분에서 이름 그대로 복사해서 붙여넣기도 해봤지만 아무리 확인해도 이름에는 문제가 없었고, 보니까 괄호 문제 였다^0^

`GetConvertTile()`의 중괄호가 닫히지 않아 그 아래에 선언한 함수들이 모두 `GetConvertTile()`내부 함수가 되었고 `Main()`에서 함수들을 인식하지 못하고 있었던것!

중괄호 하나 추가 해줬다고 모든게 해결이 됐다^~^
