## 변수
- 변수 : 프로그램에서 사용할 데이터를 임시로 저장해 놓는 그릇
- 선언 : 데이터를 담아 놓을 그릇을 만드는 작업
- 변수 이름과 값, 데이터 형식이 필요하다.

```csharp
int number;
number = 7;
```
![[Pasted image 20260209172458.png]]
### 1. 변수를 사용하는데 필요한 세가지 요소
1. 저장하고자 하는 대상의 형태
2. 저장소의 이름
3. 실제 값

### 2. 변수 선언과 동시에 초기화 하기
```csharp
static void Main()
{
	int number = 7; //변수 선언과 동시에 초기화
	Console.WriteLine("{0}, number);
}
```
`int`키워드를 사용하여 `number`라는 이름의 정수를 저장할 수 있는 변수를 만들고 값으로 7을 대입 한 후 콘솔화면에 출력

### 4. 변수명 표기법
- 헝가리안 표기법 : 변수 이름에 데이터 형식에 맞는 접두사를 붙임.

		int intNum = 0;
		
- 파스칼 표기법 : 단어를 대문자로 구분

		int MyNum = 0 ;
- 낙타 표기법(가장 선호) : 변수 이름의 첫 글자는 소문자로 시작하고, 그 다음 단어들의 첫 글자는 대문자로 시작

		int myNum = 0;
