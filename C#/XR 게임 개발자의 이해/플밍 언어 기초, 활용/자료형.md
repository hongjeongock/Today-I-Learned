### 2. 문자 데이터 형식 : char
```csharp
char charValue = 'a';
```
	
	문자형 변수는 char키워드로 선언하고 값을 초기화 할 때는 작은따옴표 2개를 사용하여 문자 하나를 묶어준다.

```csharp
static void Main()
{
	char grade = 'A';
	char kor = '가';

	Console.WriteLine(grade);
	Console.WriteLine(kor);
}
```
- 출력
> A
> 가

### 3. 문자열 데이터 형식 : string 
```csharp
stringValue = "문자열 희희"'
```

	문자열 데이터 형식의 변수를 사용 할 수 있다. 큰따옴표로 묶는다.
	
```csharp
static void Main()
{
	string name = "홍길동";
	Console.WriteLine("안녕하세요.{0}입니다.", name);
}
```
-  출력
>안녕하세요. 홍길동입니다.

#### @ 기호로 여러 줄 문자열 저장하기
```csharp
static void Main()
{
	string multiLines = @"
	안녕하세요.
	반갑습니다.
	";
	Console.WriteLine(multiiLines);
}
```
-   출력
> 안녕하세요.
> 반갑습니다.

실행하면 줄 바꿈 및 소스코드에 있는 빈 공백까지 모두 포함하여 문자열로 저장된다.
