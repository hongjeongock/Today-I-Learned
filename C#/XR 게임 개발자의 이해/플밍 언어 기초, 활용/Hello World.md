## 1. Hello World
프로그램 코드는 위에서 아래로 절차대로 실행이 된다.

```csharp
Console.WriteLine("");// 한 줄단위로 쓰는것
Console.Write(""); //글자를 쓰기만 하는것
```

공식 C# 마이크로소프트 페이지에 레이아웃 규칙과 주석달기 권고 사항. 필수사항은 아니나, 참고하면 좋음

| 레이아웃 규칙                                |
| -------------------------------------- |
| 1. 문은 한줄에 하나씩만 작성                      |
| 2. 선언은 한 줄에 하나씩만 작성                    |
| 3. 메서드 정의, 속성 정의 간에는 빈줄을 하나 이상 추가      |
| 4. 한 줄에 여러 식이 논리 결합 되어있을시, 괄호로 명확하게 구분 |

### 1. 줄 넘기기
```csharp
Console.Write("줄 바꿈 없음");
Console.WriteLine("줄 바꿈 포함");
```
- 줄 바꿈 기호 : \(역슬래시)n - 이스케이프 시퀀스
### 2. 주석(코드 설명문)
컴퓨터가 읽는 코드가 아닌 프로그래머가 읽기 위한 용도. 컴퓨터는 읽지 못한다.
```csharp
Console.WriteLine(); //슬래쉬 두개를 사용하여 주석 입력

/*Console.WriteLine(); 
여러줄에
대한
주석
작성
가능
*/
```

| 주석 달기 규칙                            |
| ----------------------------------- |
| 1. 주석은 코드의 끝이 아닌 별도의 줄에 배치          |
| 2. 영어 주석일 경우, 주석의 첫 글자는 대문자로 시작     |
| 3. 주석의 끝에는 마침표를 찍는다.                |
| 4. 코드를 위한 설명 주석일 경우, 별포 주석은 쓰지 않는다. |

### 3. 들여쓰기
프로그램 소스는 가독성을 고려하여 들여쓰기를 해야한다.
일반적으로 시작 중괄호 다음 코드는 들여쓰기를 한다.
보통 공백 4칸을 사용함(Tap키 누르면 됨)

### 4. 이스케이프 시퀀스(Escape sequnce), 확장열
문자열 안에서 쓸 수 있는 일종의 명령어로 이해하면 편하다.
\a : 벨(경고)
\n : 줄 바꿈
\t : 가로 탭
 
 줄바꿈 예시
```csharp
Console.WriteLine("내용\n");
Console.WriteLine("내용\n");
Console.WriteLine("내용");
```
가로탭 예시

```csharp
Console.WriteLine("내용\t추가텍스트") //"내용	추가텍스트" 로 한줄에 출력 됨
```

### 5. using 과 namespace

- 네임스페이스(namespace) : 클래스·구조체·인터페이스 등의 이름 충돌을 방지하기 위한 논리적 분류 체계

	- 매번 네임스페이스를 입력하면 번거롭기 때문에 using키워드를 사용하여 코드 위쪽에 `using System;`처럼 구문을 넣으면 네임스페이스를 생략하고 클래스.매서드(); 형태로 줄여서 쓸 수 있다.

```csharp 
namespace playerExtension
{

}
```

- Main()메서드: 프로그램의 시작 지점. 반드시 있어야한다. 프로그램을 실행하고 종료함.
- 중괄호{} : 프로그램 범위(스코프)를 구분
	- `{` : 열기
	- `}` : 닫기
	- `Allman 스타일` : 시작과 끝을 맞추는 스타일
		```csharp
		static void Main()
		{
			
		}
		```
	- `K & R 스타일`  :  줄의 맨 마지막에 시작 중괄호를 넣는 스타일. 가장 많이 사용.
		```csharp
		static void Main(){
		
		}
		```
- 세미콜론 (;) : 명령어의 끝
-  대소문자 구분 : C#은 대소문자를 구분한다. 정확하게 입력하지않으면 에러가 발생한다.
### 6. 자리표시자

출력기능의 심화
{n}의 형태로 {0}, {1}, {2} 순서대로 자리를 만들고 그 다음에 있는 값을 차례대로 넘겨받아 출력함.

```csharp
Console.WriteLine("{0}", "Hello, C#");
```
