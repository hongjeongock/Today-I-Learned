데이터로 연산 작업을 수행할 때는 연산자를 사용한다.
## 1. 산술 연산자

| 산술 연산자 | 의미  | 예     | 설명                      |
| :----: | --- | ----- | ----------------------- |
|   +    | 더하기 | A + B | A 와 B를 더한 값을 반환         |
|   -    | 빼기  | A - B | A에서 B를 뺀 값을 반환          |
|   *    | 곱하기 | A * B | A 와 B를 곱한 값을 반환         |
|   /    | 나누기 | A / B | A를 B로 나눈 결과를 반환         |
|   %    | 나머지 | A % B | A를 B로 나눈후 몫이 아닌 나머지값 반환 |

```csharp
Console.WriteLine(5 + 2);
Console.WriteLine(5 - 2);
Console.WriteLine(5 * 2);
Console.WriteLine(5 / 2); //몫만 출력된다.
Console.WriteLine(5 % 2); //나머지만 출력된다.
```
```csharp
Console.WriteLine("22나누기 3의 몫은 {0}", 22 / 3);
Console.WriteLine("22나누기 3의 나머지는 {0}", 22 % 3);

int firstNum = 3;
int secondNum = 9;
Console.WriteLine(firstNum + secondNum);
```
```csharp
//a = b +c;
int result;
result = firstNum * 2 + secondNum * 3 + firstNum;
```


```csharp
// 공격력에서 플레이어 방어력을 뺀 값에 보정치를 곱한 값이 최종 데미지
int playerDef = 5;
int monsterAtk = 10;
int value = 2;
//괄호를 써서 우선순위를 지정해줘야한다.
Console.WriteLine($"플레이어가 {(monsterAtk - playerDef) * value} 데미지를 입었다.");
```
## 2. 대입연산자( = 할당연산자)

| 대입연산자 | 예      | 설명                             |
| :---: | ------ | ------------------------------ |
|   =   | A = B  | A에 B 값을 대입한다.                  |
|  +=   | A += B | A에 B를 더한 후 그 값을 A에 대입한다.       |
|  -=   | A -= B | A에서 B를 뺀 후 그 값을 다시 A에 대입한다.    |
|  *=   | A *= B | A에 B를 곱한 후 그 값을 A에 다시 대입한다.    |
|  /=   | A /= B | A에서 B를 나눈 후 몫을 다시 A에 대입한다.     |
|  %=   | A %= B | A에서 B를 나눈 후 나머지 값을 다시 A에 대입한다. |

```csharp
int a = 1;
int b = 2;

a = a + b;
```

를 좀 더 간단하게

```csharp
int a = 1;
int b = 2;

a += b;
```

## 3. 증감연산자

> 변수 값을 1 증가 시키거나 감소 시키는 연산자.

- ++(증가 연산자) : 변수 값에 1을 더한다.
- --(감소 연산자) : 변수 값에 1을 뺀다

```csharp
int count = 0;

Console.WriteLine(count);
```

- 전위연산, 후위연산
	연산자가 변수 앞에 붙으면 전위 연산자, 뒤에 붙으면 후위 연산자라고 함

```csharp
int count1 = 0;
int count2= 0;

Console.WriteLine(count1++); //후위연산
Console.WriteLine(++count1); //전위연산
```

## 4. 변환 연산자
()기호를 사용하여 특정 값을 원하는 데이터 형식으로 변환할 수 있다.

```csharp
int a = 5;
int b = 6;
int c = 8;

flot average = (a + b + c) / 3;
Console.WriteLine(average);
```
6이 출력 된다.
```csharp
int a = 5;
int b = 6;
int c = 8;

flot z = (a + b + c) / (float)3;
Console.WriteLine(average);
```
6.333333 이 출력 된다. 이를 형변환이라 부른다

## 5. 비교 연산자( = 관계형 연산자)
두 항이 큰지, 작은지 또는 같은지 등을 비교한느데 사용. 참 또는 거짓으로 출력된다.

| 연산자 | 예      |    의미    |                 설명                 |
| :-: | ------ | :------: | :--------------------------------: |
|  <  | A < B  |   작은지?   |     A가 B 미만이라면 true, 아니라면 false    |
| <=  | A <= B | 작거나 같은지? |     A가 B 이하라면 true, 아니라면 false     |
|  >  | A > B  |   큰지?    |     A가 B 초과라면 true, 아니라면 false     |
| >=  | A >= B | 크거나 같은지? |    A가 B 이상이라면 true, 아니라면 false     |
| ==  | A == B |   같은지?   |  A 와 B가 같으면 True, 그렇지 않으면  False   |
| !=  | A != B |   다른지    | A 와 B가 같지 않으면 True, 그렇지 않으면  False |

```csharp
int a = 5;
int b = 66;

Console.WriteLine(a > b); // false
Console.WriteLine(a > b); // false
Console.WriteLine(a > b); // false
```
```csharp
int a = 5;
int b = 66;

bool result = a > b;

Console.WriteLine(result) ;
```
```csharp
int a = 5;
int b = 66;
bool result;

// 초과 | 미만
result = a > b;
result = a < b;

// 이상 / 이하
result = a >= b;
result = a <= b;

Console.WriteLine(result);
```

## 6. 논리 연산자

두개 이상의 bool타입 데이터에 대한 논리 연산을 수행한다.

크게는 사용 목적에 따라 Bool 논리연산자, 조건부 논리연산자로 나눠볼 수 있다.

다만, 조건부 논리연산자 쪽이 효율적으로 형태로 동작하기 때문에 자주 쓰이는 방식이다.

그럼에도 불구하고 bool 논리연산자는 이진수 타입을 연산해야 할 때도 사용되므로 알고 있어야 할 개념이다.

연산자는 기본적으로 이진 형태의 데이터, bool 타입에 대해 수행하는 것이 일반적이다.

| 연산자  |    예     | 의미  |                     설명                      |
| :--: | :------: | :-: | :-----------------------------------------: |
|  &&  |  A && B  | AND | A항과 B항 **모두** True일 때만 True, 그렇지 않으면 False  |
| \|\| | A \|\| B | OR  | A항과 B항 **모두** False일 때만 False, 그렇지 않으면 True |
|  !   |    !A    | NOT |    A항이 True이면 False, A항이 False면 True를 반환    |
- 점프
	1. 키 눌림
	2. 바닥에 붙어있어야 함
	3. 캐릭터 살아있음
	4. 다른 상태가 아닌 경우

```csharp
bool isKeyDown = true;
bool isGrounded = true;
bool isAlive = true;
bool isAnotherState = false;

bool canJump = isKeyDown && isGrounded && isAlive && ! isAnotherState;
```

>비트연산자와의 차이점은, Bool 논리연산자는 실제로 주어진 모든 데이터에 대한 논리연산을 수행하지만, 조건부 논리 연산은 이후의 논리연산을 수행할 필요가 없다면 수행하지 않는다.

## 7. 조건부 논리 연산자(비트 연산자)

| 연산자 |      의미      |             설명              |
| :-: | :----------: | :-------------------------: |
|  &  |   논리곱(AND)   | 비트 값이 둘 다 1일 때만1, 그렇지 않으면 0 |
| \|  |   논리합(OR)    | 비트 값이 하나라도 1이면 1, 그렇지 않으면 0 |
|  ^  | 배타적 논리합(XOR) | 비트 값이 서로 다를 때만 1, 그렇지 않으면 0 |
|  ~  |     NOT      |      비트 값이 1이면0, 0이면 1      |

```csharp
bool a = true;
bool b = false;
bool c = true;
bool d = false;

bool result = a | b | c | d;

Console.WriteLine(result);
```

```csharp
bool a = true;
bool b = false;
bool c = true;
bool d = false;

bool result = a & b & c & d;

Console.WriteLine(result);
```

## 8. 연산자 우선순위


| 항목     |         연산자          | 우선순위 |
| ------ | :------------------: | :--: |
| 괄호 연산자 |          ()          |  ↑   |
| 증감 연산자 |        ++, --        |  \|  |
| 산술 연산자 |          -           |  \|  |
|        |         *, /         |  \|  |
|        |          %           |  \|  |
|        |         +, -         |  \|  |
| 연결 연산자 |          +           |  \|  |
| 관계 연산자 | ==, !=, <, >, <=, >= |  \|  |
| 논리 연산자 |        !(NOT)        |  \|  |
|        |       &&(AND)        |  \|  |
|        |       \|\|(OR)       |  ↓   |
