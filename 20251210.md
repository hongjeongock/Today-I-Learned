# 반복문
## while
1. 조건식
    - false라면 바로 종료
    - true라면 로직 수행

```csharp
while(여기 내용이 참이면) 
{
    여기 내용 반복
}
```
```csharp
while(true)
{
    Console.Write("반복문 루프 도는중");
}
```
이런식으로 조건문을 작성하면 무한으로 반복이 된다.

특정 횟수만 돌리고 싶다면
```csharp
int count = 0 
int toLoop = 10;

while(count < toLoop)
{
    count++;
    Console.Write($"반복문 {count}회 루프 수행중");
}
```
짝수만 출력 하고싶을때
```csharp
int count = 0 
int toLoop = 10;

while(count < toLoop)
{
    count++;
    
    if (count % 2 == 0)
    Console.Write($"반복문 {count}회 루프 수행중");
}
```


## break, Continue
break : 루프가 즉시 종료되는 키워드
continue : 

```csharp
string input;
int inputValue;
bool isConverted = false;

while (true)
{
    COnsole.Write("제발 정수좀 입력 해주세요: ");
    input = Console.ReadLine();
    isConverted = int.TryParse(input, out inputValue);

    if(isConverted == false)
    {
        //숫자 0 입력 받으면 종료되도록 처리하고 싶을때
        if (inputValue == 0)
        {
            Console.WriteLine("0 입력, 반복문 종료");
            break;
        }
        Continue;
    }

    if (isConverted)
    {
        Console.WriteLine("정수가 입력되었습니다.");
        break;
    }
}
    Console.WriteLine("프로그램 끝");
```
# for문
