# 배열
동일한 타입의 여러 데이터 요소를 하나의 변수에 저장하고 관리하는데 사용되는 형식

서로 관련이 있고, 자료형이 같을 경우, 하나의 묶음으로 만들 수 있다. 학생들의 성적이나, 아파트의 세대주 명 등 비슷한 계열의 자료를 묶을때, 배열이라는 기능을 사용하면 편리하다.

## 다차원 배열
[,] 이차원 배열
```csharp
 Console.Clear();
     
 int[,] matrix = new int[2, 3];
 {
     { 6, 17, 8 },
     { 16, 28, 2 }
 }
```
```csharp
 Console.Clear();
     
 int[,] matrix = new int[2, 3];
 {
     { 6, 17, 8 },
     { 16, 28, 2 }
 }

 for (int i = 0; i < matrix.GetLength(dimensionL: 0); i++)
 {
     for (int j = 0; j < matrix.GetLength(dimension: 1); j++)
     {
        Console.Write("{matrix[i, j]} ");
     }
     Console.WriteLine();
```

1. 아무 입력을 요구하지 않고, 결과물 반환이 없는 형태의 함수(반환값이 없고, 인자값도 없는 형태)
```csharp
static void Main(string[] args)
{
    PrintMyName();
}
static void PrintMyName()
{
    Console.WriteLine("제 이름은");
    Console.WriteLine("홍길동");
    Console.WriteLine("입니다.");
}
```
